# AI智能截图解析工具

一个小众且实用的AI智能截图解析工具，支持上传截图、AI自动识别文字、智能解析和快捷处理功能。

## 功能特点

### 核心功能
1. **上传截图**：支持 JPG、PNG、WebP 格式的图片上传，支持Ctrl+V直接粘贴上传
2. **AI文字识别**：使用 Tesseract.js 进行 OCR 识别，支持中英文
3. **智能解析**：一键提取手机号、邮箱、网址、金额数字
4. **快捷处理**：对识别的数字做求和、平均值计算，对文字做中译英、英译中翻译
5. **结果导出**：识别的文字可一键复制，也能导出为 txt 文件

### 技术亮点
- 使用 Tesseract.js 实现浏览器端 OCR 识别
- 响应式设计，适配不同屏幕尺寸
- 模块化代码结构，易于维护和扩展
- 无后端依赖，纯前端实现，可直接部署

## 如何使用

### 本地运行
1. 克隆本仓库
2. 在浏览器中打开 `index.html` 文件
3. 上传截图并点击「开始解析」按钮
4. 查看识别结果并使用智能解析和快捷处理功能

### 部署到 GitHub Pages
1. 克隆本仓库
2. 推送代码到 GitHub 仓库
3. 在仓库设置中启用 GitHub Pages
4. 选择 `main` 分支作为源
5. 访问生成的 GitHub Pages 链接

## 技术栈

- **前端**：HTML5、CSS3、JavaScript
- **UI框架**：Tailwind CSS
- **OCR引擎**：Tesseract.js
- **翻译API**：MyMemory Translated API


## 注意事项

- OCR 识别速度取决于图片大小和浏览器性能
- 翻译功能依赖于 MyMemory Translated API，可能存在一定的延迟
- 建议使用清晰、文字大小适中的截图以获得最佳识别效果
